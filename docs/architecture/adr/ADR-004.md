# ADR-004: Anti-Repetition Check for Generated Messages

## Status

Proposed

## Context

Repeated or near-identical messages signal automation, damage trust, and may trigger Telegram spam filters.

## Decision

Compare each generated message against the last 5 outbound messages in the same conversation using Levenshtein similarity and reject outputs with similarity ≥ 0.8.

## Consequences

- Positive: reduces repetitive outbound content, supporting QR-04.
- Negative: threshold must be calibrated to avoid blocking valid follow-ups.
