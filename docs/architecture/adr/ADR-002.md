# ADR-002: Deterministic Conversation State Machine

## Status

Proposed

## Context

The sales funnel depends on predictable state transitions. Incorrect transitions corrupt analytics and can send messages to closed leads.

## Decision

Use a deterministic state machine (`app/core/state_machine.py`) with explicit transitions and immutable terminal states (`meeting_booked`, `closed`).

## Consequences

- Positive: deterministic behavior, easy to test, supports QR-02.
- Negative: new conversation states require explicit transition rules.
