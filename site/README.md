# AI Sales Manager — Промо-сайт

Самостоятельный лендинг для демонстрации проекта потенциальным заказчикам.

## Структура

- `index.html` — основная страница
- `css/styles.css` — стили
- `js/main.js` — интерактивность (меню, FAQ, анимации, форма)

## Как открыть

Локально:

```bash
cd site
python -m http.server 8080
```

Откройте `http://localhost:8080`.

## Интеграция с FastAPI

Приложение `app/main.py` монтирует эту директорию как статику по корневому URL:

```python
app.mount("/", StaticFiles(directory="site", html=True), name="site")
```

Так что после `docker-compose up` лендинг будет доступен на `http://localhost:8000/`, а API-документация — на `http://localhost:8000/docs`.

## Настройка

- Логотип, цвета и шрифты настраиваются в `css/styles.css` через CSS-переменные.
- Тексты, тарифы и контакты редактируются прямо в `index.html`.
- Форма заявки сейчас работает на фронтенде (выводит сообщение об успехе). Для подключения к backend замените обработчик в `js/main.js`.
